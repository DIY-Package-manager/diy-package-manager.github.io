Coisas a se fazer com o tempo.

Mudanças são inevitáveis, temos muito a escrever neste DIY para se tornar totalmente funcional. Para se tornar referência e atingir muitos hackers, precisamos dar um pouco do suor. Abaixo listo coisas que precisam ser feitas.

  • Traduzir todo o contéudo para outras linguagens! A internacionalização é importante.
  • Reescrever/melhorar o código implementado no livro, sempre utilizando códigos mais simples e legíveis para todos conseguirem acompanhar.
  • Formar uma comunidade e escrever este livro com ajuda. Várias cabeças pensam melhor que uma.
